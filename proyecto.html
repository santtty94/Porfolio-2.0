THE SANTY'S TOURS

Proyecto Intermodular · ASIR 2025/2026

¿Qué es?

The Santy's Tours es una plataforma de promoción y comercialización de productos turísticos en Barcelona. Nació como proyecto intermodular del ciclo ASIR, con el objetivo de simular una arquitectura e infraestructura IT real sobre un caso de uso concreto. Más allá del ámbito académico, el proyecto tiene una dimensión real: es la base sólida sobre la que, junto a un socio y compañero, se pretende ejecutar este negocio turístico en el futuro.

¿Qué problema resuelve?

Centraliza la gestión de clientes, rutas y reservas de una empresa de turismo, permitiendo administrar toda la infraestructura IT desde un único sistema: servidores, base de datos, monitorización y acceso remoto. Todo ello diseñado para que sea escalable y pueda trasladarse a un entorno de producción real.

¿Para quién está pensado?

Para una empresa de turismo que necesita gestionar su infraestructura de forma segura, escalable y accesible en la nube. En este caso, orientado específicamente al mercado turístico de Barcelona y sus productos de tour.

Tecnologías utilizadas

Linux · Windows Server · VirtualBox · AWS · SQL · HTML/CSS · DNS · DHCP · SSH · PowerShell

¿Qué he aprendido?

Este proyecto me ha enseñado a pensar en sistemas completos, no en piezas aisladas. He aprendido a levantar y conectar servidores, gestionar bases de datos reales, administrar servicios en la nube y documentar todo el proceso de forma profesional. También he aprendido a tomar decisiones técnicas sin supervisión directa y a aplicar conocimientos que aún no estaban en el temario cuando el proyecto los requería.
